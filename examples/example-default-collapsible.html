This is a demo for the Expounder concept, with the default set to making all links collapsible. A concept is an experimental model that is designed to test viability of a product. Expounder is meant to be implemented in tutorials and other explanatory texts, where there are readers of disparate levels of experience on the topic. Levels of experience are when someone knows more things than someone else.

If a reader needs clarification on a term, clicking the link will expand a piece of text, image or other elements. Elements can also be infinitely nested (i.e. elements can contain other elements like this).

This content can be collapsed by clicking on the link again.
